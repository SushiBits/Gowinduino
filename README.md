# Gowinduino

Gowinduino 是一款基于高云小蜜蜂 GW1NSR-4C 硬处理器 SoC FPGA 的试验开发板。

## 特性

GW1NSR-4C FPGA 有如下特性：

* 4608 个 LUT
* 3456 个寄存器
* 10 块 18kbit SRAM
* 256kbit 用户闪存
* 64Mbit PSRAM SiP
* 内置 Cortex-M3 硬 IP

本开发版有如下特性：

* 最高 12V 外部 DC 输入
* 3.3V I/O 电压
* Arduino Uno 脚位
* 集成 FT2232HL JTAG 和串口接口
